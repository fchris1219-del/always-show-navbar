# 悬浮导航轮盘（SillyTavern / 云酒馆扩展）

一个独立的悬浮球：点击展开成**轮盘**，快速跳转到酒馆各大功能面板，或一键回到聊天。

专为「顶部导航栏在窄屏/云端会消失」的场景设计——不去强改布局，而是提供一个
始终可用的悬浮入口。

## 特点

- **自适应**：轮盘节点在每次打开时实时扫描酒馆真实的顶部图标生成，
  自动兼容「酒馆菜单精简器」等改过菜单的酒馆。
- **不依赖导航栏可见**：顶部图标即使被隐藏，也能通过轮盘触发。
- **不破坏布局**：组件完全独立（`#nvw-*` 命名空间 + `position: fixed`），
  不修改任何原有元素。
- 悬浮球**可拖动**并记忆位置；含「回到聊天」节点（关闭所有面板）。

## 用法

- 单击悬浮球：展开/收起轮盘
- 点轮盘节点：打开对应功能面板
- 拖动悬浮球：移动位置（自动记忆）
- `Esc` 或点击空白处：收起

## 安装

酒馆 → `扩展` → `安装扩展`，粘贴本仓库 Git 地址；或把文件夹放到
`data/<用户名>/extensions/`（新版）/ `public/scripts/extensions/third-party/`（旧版）。
